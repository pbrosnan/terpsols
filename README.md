# terpsols
Solutions for UMD Mathematics formalized in Lean

# Project Goals 
The goal of this project is to learn some 
[Lean](https://lean-lang.org)
 by 
formalizing UMD Mathematics Department Exams (and, possibly later, 
some homework as well).
The starting point is to formalize the Fall 2025 Graduate Qualifying
Exams.

# Useful Links

- [Qualifying Exams](https://www-math.umd.edu/quals.html)
- [Final Exam Testbank](https://www-math.umd.edu/testbank.html)
- [Mathematics in Lean4](https://leanprover-community.github.io/mathematics_in_lean/)


## GitHub configuration

To set up the GitHub repository, follow these steps:

* Under your repository name, click **Settings**.
* In the **Actions** section of the sidebar, click "General".
* Check the box **Allow GitHub Actions to create and approve pull requests**.
* Click the **Pages** section of the settings sidebar.
* In the **Source** dropdown menu, select "GitHub Actions".

After following the steps above, you can remove this section from the README file.
